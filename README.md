# luckybot
> Opinionated Instagram automation tool


### Install

```
$ yarn add luckybot
```

### Development

**install dependencies**

```
$ yarn
```

**run dev mode**

```
$ LUCKYBOT_USER=your_instagram_user LUCKYBOT_PASS=your_instagram_password LUCKYBOT_HASTAG=hashtag_to_like yarn dev
```

**check types**

```
$ yarn typecheck
```

**build**

```
$ yarn build
```
